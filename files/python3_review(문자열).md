## Python3 Review

### Python3 Basic

1. 자료형

   - 숫자

     - 파이썬의 숫자는 크게 정수, 실수 2가지로 나눌 수 있다. 

       ```python
       # 파이썬에서는 자바스크립트의 var, const, let이나 c계열 언어의 int, str, float등과 같은 지시어를 사용하지 않는다.
       a = 100
       a = -100
       a = 0
       # 위와 같이 단순히 변수를 선언하고 값을 할당하면 변수에 값이 저장된다.
       # 변순에는 다양한 값을 담을 수 있는데 위와 같이 정수를 할당하는 경우도 있고 실수를 할당 할 수 도 있다.
       a = 1.2
       a = -3.45
       # 단 위와 같이 같은 이름의 변수에 값을 재할당하게 되면 코드는 위에서 부터 해석하기 때문에 가장 밑에 있는 변수의 값이 a 라는 변수에 저장되므로 a를 출력하면 -3.45가 출력 되게 된다.
       
       # 파이썬에서는 단순한 사칙연산을 시작으로 다양한 계산이 가능하다.
       a = 5
       b = 5
       c = a + b 
       # a와 b에 각각 5라는 값을 할당하여 a, b를 더한 값을 c에 할당 했다. c는 10이라는 값이 출력된다.
       # +말고도 위에서 언급한 것처럼 -, *, /도 가능하다.
       
       # 프로그래밍에서는 사칙연산에 사용되는 기본 연산자 이외에도 특별한 연산자들이 있다.
       a = 2
       b = 3
       a ** b 
       # 두 값의 제곱을 나타내는 **연산자 (8 출력)
       a % b
       # 두 값을 나누고 그 나머지를 반환하는 % 연산자 (1 출력)
       a // b
       # 두 값을 나누고 그 몫을 반환하는 // 연산자 (0 출력)
       ```

   - 문자열

     - 문자열이란 문자, 단어등으로 구성된 문자들의 집합을 의미한다.

       ```python
       # 문자열을 만드는 방법은 간단하다. 사용자가 원하는 문자나 단어를 '' 작은 따옴표나, "" 큰 따옴표안에 문자나 단어를 넣으면 된다.
       "hello world"
       'hello world'
       # 큰 따옴표를 """ """ 3개로 문자를 둘러 쌓거나 작은 따옴표 ''' ''' 3개를 사용하는 것도 가능하다.  
       """need for speed"""
       '''show me the money'''
       
       # 원하는 문자열을 변수에 할당하면 간단하게 문자열 값을 저장할 수 있다.
       a = 'hello'
       b = 'world'
       
       # 파이썬은 간단한 문자열 연산자를 허용한다. 타 언어에는 없는 특별한 기능이다.
       a + b
       'hello world'
       a * 2
       'hellohello'
       
       # 파이썬 내장함수를 사용하여 문자열의 길이를 쉽게 구할 수 있다.
       len(a)
       5
       
       # 문자열 포매팅을 사용하여 원하는 문자열을 변수와 함께 사용하여 만들 수 있다.
       "I need %d won." % 10000
       "I need 10000 won."
       ```

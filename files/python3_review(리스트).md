## Python3 Review

### Python3 Basic

1. 자료형

   - 리스트

     - 리스트는 다양한 값을 하나의 변수로 만들어 목록(리스트)처럼 사용 할 수 있게 해주는 자료형 이다.

     ```python
     # 리스트는 다양한 값을 담아서 저장 할 수 있다.
     a = list()
     a = []
     a = [1, 2, 3]
     
     # 위와 같이 아무 값도 들어 있지 않는 a 리스트를 만들 수 있으며 그 외에 여러 값을 넣어서 저장 할 수도 있다. 리스트는 다양한 자료형을 구분 없이 묶어 둘 수 있는 특징이 있다.
     
     a = [1, 'hello', 1.4, -1.2, 'world']
     
     # 리스트는 인덱싱을 사용하여 안에 들어 있는 특정한 값을 꺼내서 사용 할 수 있다.
     a = [1, 2, 3, 4, 5]
     a[0]
     1
     
     # 리스트 자료형에 [인덱싱할 숫자]를 집어 넣으면 위와 같이 값을 꺼낼 수 있다. 프로그래밍 특징상 숫자는 0번 부터 센다.
     
     a[-1]
     5
     
     # 파이썬에서는 음수를 사용하여 인덱싱 할 경우 뒤에서부터 값을 꺼내서 사용 할 수도 있다.
     
     a[0:3]
     1,2,3
     
     # 리스트에서는 인덱싱외에도 슬라이싱을 사용하여 값을 꺼낼 수도 있다. 슬라이싱은 :를 사용하여 범위를 설정하며 : 뒤에 오는 숫자보다 하나 적은 만큼 값을 꺼내게 된다.
     
     # 리스트는 문자열과 마찬가지로 간단한 연산이 가능하다.
     a = [1, 2, 3]
     b = [4, 5, 6]
     a + b
     [1, 2, 3, 4, 5, 6]
     a * 2
     [1, 2, 3, 1, 2, 3]
     
     # 리스트와 문자열은 매우 비슷하다. 문자열의 len()함수처럼 리스트의 길이도 len()로 구한다.
     len(a)
     3
     수
     # 리스트는 다양하고 유용한 함수를 제공한다.
     
     # 리스트에 새로운 값을 추가하는 함수
     a = [1, 2, 3]
     a.append(4)
     a
     [1, 2, 3, 4]
     
     # 리스트의 값을 내림차순으로 정리하는 함수
     a = [1, 4, 2, 3]
     a.sort()
     a
     [1, 2, 3, 4]
     
     # 리스트의 값을 뒤집는 함수
     a.reverse()
     a
     [4, 3, 2, 1]
     
     # index의 위치를 반환하는 함수
     a = [1, 2, 3, 4]
     a.index(3)
     2
     
     # 리스트의 요소를 제거하는 함수 요소의 모든 값을 제거한다.
     a = [1, 2, 3, 1, 2, 3]
     a.remove(2)
     a
     [1, 3, 1, 3]
     
     # 리스트의 맨 마지막 요소를 반환하고 그 요소는 제거하는 함수
     a = [1, 2, 3]
     a.pop()
     3
     a
     [1, 2]
     
     # 리스트의 특정 요소가 몇개 있는지 개수를 반환해주는 함수
     a = [1, 2, 3, 1, 2, 3]
     a.count(1)
     2
     
     # 리스트를 확장하는 함수, 리스트 연산으로 해주는 효과와 동일하다.
     a = [1, 2]
     b = [3, 4]
     a.extend(b)
     a
     [1, 2, 3, 4]
     
     ```
